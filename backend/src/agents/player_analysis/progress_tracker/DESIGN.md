# ProgressTrackerAgent Design Document

## Overview
Tracks and quantifies player improvement over recent patches (10-20 versions) with focus on identifying growth trends and breakthrough moments.

## Key Features
- **Temporal comparison**: Early half vs late half performance
- **Skill progression**: Core metrics improvement tracking (winrate, KDA, participation)
- **Learning velocity**: Speed of adaptation to new champions/roles
- **Breakthrough detection**: Identifying significant improvement milestones
- **Stability analysis**: Performance variance reduction over time

## Input: 10-20 recent packs | Output: 2000-2500 word progress report + improvement score
