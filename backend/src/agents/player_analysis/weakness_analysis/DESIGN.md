# WeaknessAnalysisAgent Design Document

## Overview
Diagnoses player weaknesses across multiple dimensions (laning, teamfight, macro) and provides prioritized improvement recommendations.

## Key Features
- **Lane phase analysis**: CS, early deaths, laning pressure
- **Teamfight analysis**: Positioning, damage output, survival
- **Macro play analysis**: Objective control, map awareness
- **Champion pool gaps**: Missing meta champions
- **Prioritized recommendations**: Top 3-5 actionable improvements

## Input: Last 5-10 packs | Output: 1500-2000 word diagnostic report + priority list
